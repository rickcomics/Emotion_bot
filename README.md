# Emotion Bot

Telegram‑бот для анализа эмоций по методу «Ситуация → Чувства → Мысли → Действия».

## Функционал

- Сбор описания ситуации.
- Выбор базовых эмоций и подэмоций из списка.
- Запись мыслей и действий.
- Формирование сводки и отправка пользователю.
